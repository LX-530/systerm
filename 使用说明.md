## 💡 如何重新运行分析

如果数据更新，可以重新运行分析：

```bash
# 确保已安装必要的库
pip3 install pandas matplotlib seaborn openpyxl

# 将新的Excel文件命名为 程宇昕.xlsx 放在同一目录

# 运行分析脚本
python3 analyze_products.py
```

分析将自动生成新的报告和图表。
